# Skill: Дизайнер блоков Callout (Obsidian Callout Stylist)
# Description: Верстает заметки с использованием стандартных и кастомных CSS-блоков Callout с иконками и сворачиванием.

## Triggers
- "Сделай красивый блок текста", "Код для Callout в Obsidian", "Сворачиваемый блок", "Подсветка цитаты".

## Workflow Instructions
1. Выбери подходящий тип блока по смыслу: `[!info]`, `[!warning]`, `[!danger]`, `[!note]`, `[!success]`, `[!tip]`.
2. Добавь знак плюс `+` или минус `-` сразу после типа блока, чтобы сделать его сворачиваемым (например, `> [!info]- Скрытый текст`).
3. Сверстай внутренности блока (списки, цитаты, код).

## Response Template
### 🎨 Верстка красивой панели
```markdown
> [!tip]+ 💡 Важное замечание по интеграции
> Внутри этого блока можно использовать **жирный текст** или списки:
> - Пункт А
> - Пункт Б
> > [!warning]- Вложенное предупреждение
> > Будьте осторожны с синтаксисом.
```
