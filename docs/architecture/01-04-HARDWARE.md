# Железо и распределение

Спецификации hardware и распределение ресурсов. Архитектура КРЕПОСТЬ рассчитана на работу целиком на локальном железе без зависимости от облачных GPU.

## Основные вычислительные узлы

### Mac Studio M4 Max 64GB RAM / 1TB SSD

Основной боевой сервер. Выполняет все production-задачи:
- **AI-инференс:** Qwen3.6-27B (Q4_K_M квантизация) — основная генеративная модель
- **Guard:** Qwen3Guard-Gen-4B — модель безопасности, классификация запросов
- **Эмбеддинги:** BGE-M3 — мультиязычная модель для векторного поиска
- **RAG-пайплайн:** ChromaDB + гибридный поиск (Vector + BM25 + Reranker)

64GB unified memory позволяет держать в памяти основную модель (~16GB в Q4_K_M) и Guard (~3GB) одновременно, с запасом для RAG-индексов и кеша.

### MacBook Air M5 32GB RAM / 1TB SSD

Выделенный узел для adversarial training (грязная зона):
- **Attacker:** Mistral-7B — генерация adversarial-промптов для тренировки Guard
- Тренировочные пайплайны и разметка
- Физически изолирован от боевой системы (Камень 1)

32GB достаточно для инференса Mistral-7B в квантизированном формате и параллельной работы тренировочных скриптов.

## Хранилище

### WD SN850X 2TB — Thunderbolt 5 (80Gbps)

Съёмный SSD для быстрого хранилища:
- Тренировочные данные, adversarial-примеры, яды
- Разметка и метаданные тренировки
- Thunderbolt 5 обеспечивает скорость, достаточную для real-time работы с вектор-индексами

Подключение/отключение этого диска переключает систему между режимами тренировки и боевой эксплуатации.

### 6TB Archive HDD

Архивное хранилище для:
- Исторических данных и бэкапов
- Полных аудит-логов (audit_hash + trace_hash)
- Архивных версий вектор-индексов

## Мониторинг и алерты

### iPad Air M4 256GB

Используется как панель мониторинга:
- Дашборд состояния системы в реальном времени
- Визуализация метрик Guard (false positive rate, latency)
- Мониторинг RAG-запросов и качества ответов

### iPhone Air 256GB

Канал оперативных алертов:
- **Pushover** — мгновенные push-уведомления о критических событиях
- **Telegram** — алерты и отчёты о состоянии системы
- Уведомления при срабатывании Guard, аномалиях в трафике, ошибках системы
