# Физическая изоляция

Описание физической изоляции компонентов системы. Архитектура КРЕПОСТЬ построена на принципе физического разделения боевой и тренировочной сред, что исключает случайное или преднамеренное проникновение adversarial-данных в production.

## Mac Studio (боевая система)

Mac Studio M4 Max — основной вычислительный узел, на котором работают:
- **Основная AI-модель** (Qwen3.6-27B) — генерация ответов
- **Guard-модель** (Qwen3Guard-Gen-4B) — фильтрация и оценка безопасности
- **RAG-пайплайн** — поиск по базе знаний (ChromaDB + BGE-M3 эмбеддинги)

Mac Studio является стерильной системой. На ней никогда не хранятся adversarial-примеры, яды или тренировочные данные атакующей модели.

## MacBook Air (грязная зона)

MacBook Air M5 используется исключительно для adversarial training:
- **Mistral-7B attacker** — генерация adversarial-промптов для тренировки Guard
- Тренировочные пайплайны для усиления защиты
- Разметка и подготовка тренировочных данных

Грязная зона намеренно изолирована от боевой системы. Данные из неё попадают в боевую систему только в виде эмбеддингов (Камень 2).

## Съёмный SSD (WD SN850X 2TB)

Съёмный SSD служит физическим носителем для:
- Тренировочных данных и adversarial-примеров
- Ядов (poisoned prompts) для обучения Guard
- Разметки и метаданных тренировки

**Принцип работы:** диск воткнут = режим тренировки, диск вынут = стерильная боевая система. Это простейший и наиболее надёжный механизм air-gap: физическое отсутствие носителя гарантирует отсутствие данных.

## Сетевая изоляция

- **Tailscale** используется для сетевой изоляции между узлами. Только авторизованные устройства в Tailscale-сети имеют доступ к API и сервисам КРЕПОСТИ.
- Прямой доступ из интернета к боевой системе отсутствует.

## Облачный карантин

Любые данные, поступающие из облачных источников, обязательно проходят через карантинный процесс:
- Все облачные данные помечаются флагом `quarantine: true`
- Карантинные данные не используются до прохождения полной проверки через Guard
- Только после верификации данные переходят в статус `trusted` и становятся доступны для RAG-пайплайна
